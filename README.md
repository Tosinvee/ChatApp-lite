ChatAppLite

A minimal real-time + push notification demo app built to understand how authentication, device registration, WebSockets, Redis presence, and Firebase push notifications work together.

This project is not just about chat features — it is about notification delivery correctness.

Tech Stack

NestJS – Backend framework

MongoDB (Mongoose) – Persistent storage

Redis – Real-time presence (online users)

Socket.IO – WebSocket communication

Firebase Cloud Messaging (FCM) – Push notifications
